# 更新日志

记录 Datelife 每个版本的变更内容。

---

## [1.0.0] - 2026-05-18

### 新增

- 食品管理核心功能 — 录入、编辑、删除食品
- 自动过期状态计算 — 可食用（>14天）/ 临期（≤14天）/ 已过期
- 二维码生成 — 每个食品生成专属二维码，扫码直达详情页
- 响应式双视图布局
  - PC 端：表格视图，宽屏管理面板
  - 手机端：卡片列表，紧凑单手操作
  - 详情页：PC 左右分栏 / 手机上下堆叠
- 食品详情页 — 完整信息展示 + 可打印二维码
- 删除确认弹窗，防止误操作

### 技术栈

- 前端：Vue 3 + Vite + Tailwind CSS + Vue Router
- 后端：Express.js + better-sqlite3 (SQLite)
- 二维码：node-qrcode
